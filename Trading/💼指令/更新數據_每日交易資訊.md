# Google Sheet 交易資料欄位

從第2列開始寫入，不可覆蓋第1列標題列。

- A欄: 交易日期，帶入 {date} 資料，格式為 YYYY/MM/DD 。
- B欄: 開盤，帶入 {open} 資料，格式為千分位及小數點後2位的數字，不帶錢字符號。
- C欄: 最高，帶入 {max} 資料，格式為千分位及小數點後2位的數字，不帶錢字符號。
- D欄: 最低，帶入 {min} 資料，格式為千分位及小數點後2位的數字，不帶錢字符號。
- E欄: 收盤，帶入 {close} 資料，格式為千分位及小數點後2位的數字，不帶錢字符號。
- F欄: 漲跌，帶入 {spread} 資料，格式為千分位及小數點後2位的數字，不帶錢字符號。
- G欄: 成交股數，帶入 {Trading_Volume} 資料後除以1000再四捨五入，格式為千分位整數的數字，不帶錢字符號。
- H欄: 外資，帶入 {name} 為 Foreign_Investor 和 Foreign_Dealer_Self 的 {buy} 減 {sell} 的總計除以1000再四捨五入，格式為千分位整數的數字，不帶錢字符號。
- I欄: 投信，帶入 {name} 為 Investment_Trust 的 {buy} 減 {sell} 的總計除以1000再四捨五入，格式為千分位整數的數字，不帶錢字符號。
- J欄: 自營，帶入 {name} 為 Dealer_self 和 Dealer_Hedging 的 {buy} 減 {sell} 的總計除以1000再四捨五入，格式為千分位整數的數字，不帶錢字符號。
- K欄: 法人合計，為「H欄」和「I欄」和「J欄」的數字加總。
- L欄: 融資餘額，帶入 {MarginPurchaseTodayBalance} 資料。
- M欄: 融券餘額，帶入 {ShortSaleTodayBalance} 資料。
- N欄: 券資比%，為「M欄」除以「L欄」乘以100%。
- O欄: 本益比，帶入 {PER} 資料。
- P欄: 帶入下方 `# Google Sheet 公式` 的 `## P欄公式` 。
- Q欄: 帶入下方 `# Google Sheet 公式` 的 `## Q欄公式` 。
- R欄: 帶入下方 `# Google Sheet 公式` 的 `## R欄公式` 。
- S欄: 帶入下方 `# Google Sheet 公式` 的 `## S欄公式` 。
- T欄: 帶入下方 `# Google Sheet 公式` 的 `## T欄公式` 。
- U欄: 帶入下方 `# Google Sheet 公式` 的 `## U欄公式` 。
- V欄: 帶入下方 `# Google Sheet 公式` 的 `## V欄公式` 。
- W欄: 帶入下方 `# Google Sheet 公式` 的 `## W欄公式` 。

# FinMind API

以下是個股每日資訊的 JSON API 路徑及欄位說明。

每個 API 路徑的 {stock_id} 參數輸入指定的股票代號，至於 {YYYY-MM-DD} 參數則表示撈取從指定日期的當月份第一天直到今天的資料，例如 https://api.finmindtrade.com/api/v4/data?dataset=TaiwanStockPrice&data_id=2330&start_date=2025-10-23 路徑意思是「2330 台積電」從2025年10月開始一直到今天的每日股價。

每次執行時先檢查 Google Sheet 裡面的最新交易日，以該交易日作為 {YYYY-MM-DD} 日期參數。如果目前尚未存在任何一筆資料，則以 2020-01-01 作為 {YYYY-MM-DD} 日期參數。

## 交易資訊

API 路徑: https://api.finmindtrade.com/api/v4/data?dataset=TaiwanStockPrice&data_id={stock_id}&start_date={YYYY-MM-DD}

- date: 交易日期。
- stock_id: 股票代號。
- open: 開盤價。
- max: 最高價。
- min: 最低價。
- close: 收盤價。
- spread: 漲跌價差。
- Trading_Volume: 成交股數。

## 三大法人明細

API 路徑: https://api.finmindtrade.com/api/v4/data?dataset=TaiwanStockInstitutionalInvestorsBuySell&data_id={stock_id}&start_date={YYYY-MM-DD}

- date: 交易日期。
- stock_id: 股票代號。
- name: 法人類型，參考 `### 法人類型` 的說明，表示此筆資料為該法人類型的買賣明細。
- buy: 買進股數。
- sell: 賣出股數。

### 法人類型

- Foreign_Investor: 外資及陸資。
- Foreign_Dealer_Self: 外資自營商。
- Investment_Trust: 投信。
- Dealer_self: 自營商自行買賣。
- Dealer_Hedging: 自營商避險。

## 融資融券

API 路徑: https://api.finmindtrade.com/api/v4/data?dataset=TaiwanStockMarginPurchaseShortSale&data_id={stock_id}&start_date={YYYY-MM-DD}

- date: 交易日期。
- stock_id: 股票代號。
- MarginPurchaseTodayBalance: 融資餘額。
- ShortSaleTodayBalance: 融券餘額。

## 本益比

API 路徑: https://api.finmindtrade.com/api/v4/data?dataset=TaiwanStockPER&data_id={stock_id}&start_date={YYYY-MM-DD}

- date: 交易日期。
- stock_id: 股票代號。
- dividend_yield: 現金殖利率。
- PER: 本益比。
- PBR: 股價淨值比。

	COUNTA($D2:$G2) = 0,

		N, '儀表板'!$A$4,

				NOT(ISNUMBER($E2)),
