# 任務

除了執行任務，也務必遵守 `# 額外規範` 的規定。

1. 透過 [[Trading/⚙️參數設定/rosy-zoo-447904-j1-a600c9e990ca.json]] API 進入 [股票名單](https://docs.google.com/spreadsheets/d/1MuJgPwiJpjyU8LXevCxUKsbsLPpMT7H9J2wnPcVqIpE/edit?gid=1951214900#gid=1951214900) ，當該列的J欄為 true 時，為該列的 stock_id 收集 `# 重點情報` 提到的資料。例如當 J3 為 true 而 J4 為 false 時，只要收集第3列 stock_id 的情報。

2. 將收集到的情報資料，依照 stock_id 分類後儲存於 [[Trading/⌚️暫存/for_python/business_database.json]] 檔案內。

3. 將 [[Trading/⌚️暫存/for_python/business_database.json]] 檔案內的資料，依照 `# Google Sheet 欄位` 分類擺放，供後續上傳 Google Sheet 使用。

4. 透過 [[Trading/⚙️參數設定/rosy-zoo-447904-j1-a600c9e990ca.json]] API 將整理好的 [[Trading/⌚️暫存/for_python/business_database.json]] 資料，上傳到 [股票名單](https://docs.google.com/spreadsheets/d/1MuJgPwiJpjyU8LXevCxUKsbsLPpMT7H9J2wnPcVqIpE/edit?gid=1951214900#gid=1951214900) 對應的 stock_id 並且該列J欄為 True 的 F欄（google_sheet_business） 欄位。

# 額外規範

1. 禁止使用「---」分隔線。
2. 每個段落都要從 ## 這種 <h2> 開始使用，也就是說大段落為 <h2>，裡面如果還有子段落，則依序為 <h3> 和 <h4>。
3. 如果 [股票名單](https://docs.google.com/spreadsheets/d/1MuJgPwiJpjyU8LXevCxUKsbsLPpMT7H9J2wnPcVqIpE/edit?gid=1951214900#gid=1951214900) 對應的 google_sheet_business 欄位是「xxx」則略過該檔股票。
