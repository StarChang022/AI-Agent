# Google Sheet 交易資料欄位

從第2列開始寫入，不可覆蓋第1列標題列。

- A欄: 交易日期，帶入 {date} 資料，格式為 YYYY/MM/DD 。
- B欄: 開盤，帶入 {open} 資料，格式為千分位及小數點後2位的數字，不帶錢字符號。
- C欄: 最高，帶入 {max} 資料，格式為千分位及小數點後2位的數字，不帶錢字符號。
- D欄: 最低，帶入 {min} 資料，格式為千分位及小數點後2位的數字，不帶錢字符號。
- E欄: 收盤，帶入 {close} 資料，格式為千分位及小數點後2位的數字，不帶錢字符號。
- F欄: 漲跌，帶入 {spread} 資料，格式為千分位及小數點後2位的數字，不帶錢字符號。
- G欄: 成交金額，帶入 {Trading_money} 資料後除以100000000再四捨五入，格式為千分位整數的數字，不帶錢字符號。
- H欄: 帶入下方 `# Google Sheet 公式` 的 `## H欄公式` 。
- I欄: 帶入下方 `# Google Sheet 公式` 的 `## I欄公式` 。
- J欄: 帶入下方 `# Google Sheet 公式` 的 `## J欄公式` 。
- K欄: 帶入下方 `# Google Sheet 公式` 的 `## K欄公式` 。
- L欄: 帶入下方 `# Google Sheet 公式` 的 `## L欄公式` 。
- M欄: 帶入下方 `# Google Sheet 公式` 的 `## M欄公式` 。
- N欄: 帶入下方 `# Google Sheet 公式` 的 `## N欄公式` 。
- O欄: 帶入下方 `# Google Sheet 公式` 的 `## O欄公式` 。
- P欄: 帶入下方 `# Google Sheet 公式` 的 `## P欄公式` 。

# FinMind API

以下是櫃買指數每日資訊的 JSON API 路徑及欄位說明。

每個 API 路徑的 {YYYY-MM-DD} 表示撈取從指定日期的當月份第一天直到今天的資料，例如 https://api.finmindtrade.com/api/v4/data?dataset=TaiwanStockPrice&data_id=TPEx&start_date=2020-01-01 路徑意思是「櫃買指數」從2020年01月開始一直到今天的每日指數。

每次執行時先檢查 Google Sheet 裡面的最新交易日，以該交易日作為 {YYYY-MM-DD} 日期參數。如果目前尚未存在任何一筆資料，則以 2020-01-01 作為 {YYYY-MM-DD} 日期參數。

## 交易資訊

API 路徑: https://api.finmindtrade.com/api/v4/data?dataset=TaiwanStockPrice&data_id=TPEx&start_date={YYYY-MM-DD}

- date: 交易日期。
- stock_id: 股票代號，此處固定為 TPEx 。
- open: 開盤指數。
- max: 最高指數。
- min: 最低指數。
- close: 收盤指數。
- spread: 漲跌指數差。
- Trading_money: 成交金額。

## H欄公式

=IF(
	COUNTA($D2:$G2) = 0, 
	"", 
	LET(
		N, '儀表板'!$A$4,
		range, OFFSET($E2, 0, 0, N, 1),
		IF(
			OR(
				NOT(ISNUMBER($E2)),
				COUNT(range) < N
			),
			"-",
			AVERAGE(range)
		)
	)
)
