# Google Sheet 交易資料欄位

從第2列開始寫入，不可覆蓋第1列標題列。

- A欄: 該月份代表日期，帶入 {date} 資料，格式為 YYYY/MM/DD 。
- B欄: 年份，帶入 {revenue_year} 資料，
- C欄: 月份，帶入 {revenue_month} 資料，
- D欄: 營收億，帶入 {revenue} 資料後除以 100000000 再四捨五入，格式為千分位及小數點後2位的數字，不帶錢字符號。
- E欄: 月增率%，帶入下方 `# Google Sheet 公式` 的 `## E欄公式` 。
- F欄: 年增率%，帶入下方 `# Google Sheet 公式` 的 `## F欄公式` 。

# FinMind API

以下是個股每月營收的 JSON API 路徑及欄位說明。

每個 API 路徑的 {stock_id} 參數輸入指定的股票代號，至於 {YYYY-MM-DD} 參數則
用於指定起始月份的第一天，表示撈取從指定日期的當月份直到今天的資料，例如 https://api.finmindtrade.com/api/v4/data?dataset=TaiwanStockPrice&data_id=2330&start_date=2020-01-01 路徑意思是「2330 台積電」從2020年01月（認列2019年12月營收）開始一直到今天有公布的每月營收。

必須注意 {YYYY-MM-DD} 參數都要是該月第一天，例如 2020-01-01 然後是 2020-02-01 再來是 2020-03-01 這樣依此類推。

每次執行時先檢查 Google Sheet 裡面的最新交易日，以該交易日作為 {YYYY-MM-DD} 日期參數。如果目前尚未存在任何一筆資料，則以 2020-01-01 作為 {YYYY-MM-DD} 日期參數。

## 每月營收

API 路徑: https://api.finmindtrade.com/api/v4/data?dataset=TaiwanStockMonthRevenue&data_id={stock_id}&start_date={YYYY-MM-DD}

- date: 代表日期，抓取代表日期的上月營收，例如 2025-01-01 的會抓到 revenue_year: 2024 和 revenue_month: 12 的營收。
- revenue: 營收。
- revenue_month: 月份。
- revenue_year: 年份。

# Google Sheet 公式

以下皆為第2列的範例公式，所以公式內的列數需根據第幾列自動替換。例如 `## D欄公式` 如果放在第8列則為「=IF(COUNTA($C8) = 0, "", IF(OR(NOT(ISNUMBER($C9)), $C9 = 0), "-", ($C8 - $C9) / $C9))」、例如 `## E欄公式` 如果放在第26列則為「=IF(COUNTA($C26) = 0, "", IF(OR(NOT(ISNUMBER($C38)), $C38 = 0), "-", ($C26 - $C38) / $C38))」。

## E欄公式

=IF(COUNTA($D2) = 0, "", IF(OR(NOT(ISNUMBER($D3)), $D3 = 0), "-", ($D2 - $D3) / $D3))

## F欄公式

=IF(COUNTA($D2) = 0, "", IF(OR(NOT(ISNUMBER($D14)), $D14 = 0), "-", ($D2 - $D14) / $D14))
