# 任務

除了執行任務，也務必遵守 `# 額外規範` 的規定。

2. 將收集到的情報資料，依照 stock_id 分類後儲存於 [[Trading/⌚️暫存/for_python/business_database.json]] 檔案內。

# 額外規範

1. 禁止使用「---」分隔線。
2. 每個段落都要從 ## 這種 <h2> 開始使用，也就是說大段落為 <h2>，裡面如果還有子段落，則依序為 <h3> 和 <h4>。
3. 如果 [關注名單](https://docs.google.com/spreadsheets/d/1MuJgPwiJpjyU8LXevCxUKsbsLPpMT7H9J2wnPcVqIpE/edit?gid=1951214900#gid=1951214900) 對應的 google_sheet_business 欄位是「xxx」則略過該檔股票。
