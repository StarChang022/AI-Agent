# FinMind API

以下是個股財報數據的 JSON API 路徑及欄位說明。

每個 API 路徑的 {stock_id} 參數輸入指定的股票代號，至於 {YYYY-MM-DD} 參數則表示撈取從指定日期的當月份第一天直到今天的資料，例如 https://api.finmindtrade.com/api/v4/data?dataset=TaiwanStockBalanceSheet&data_id=2330&start_date=2020-01-01 路徑意思是「2330 台積電」從2020年01月開始一直到今天的每季財報。

每次執行時先檢查 Google Sheet 裡面的最新日期，以該日期作為 {YYYY-MM-DD} 日期參數。如果目前尚未存在任何一筆資料，則以 2020-01-01 作為 {YYYY-MM-DD} 日期參數。

## 資產負債表

API 路徑: https://api.finmindtrade.com/api/v4/data?dataset=TaiwanStockBalanceSheet&data_id={stock_id}&start_date={YYYY-MM-DD}

- date: 日期。
- stock_id: 股票代號。
- type: 數據類型，參考 `### 資產負債表數據類型` 的說明，表示此筆資料為該數據類型的數據。
- value: 值。

### 資產負債表數據類型

- CashAndCashEquivalents: 現金及約當現金。
- AccountsReceivableNet: 應收帳款淨額。
- Inventories: 存貨。
- CurrentAssets: 流動資產合計。
- PropertyPlantAndEquipment: 不動產、廠房及設備。
- NoncurrentAssets: 非流動資產合計。
- TotalAssets: 資產總額。
- AccountsPayable: 應付帳款。
- CurrentLiabilities: 流動負債合計。
- LongtermBorrowings: 長期借款。
- BondsPayable: 應付公司債。
- NoncurrentLiabilities: 非流動負債合計。
- Liabilities: 負債總額。
- RetainedEarnings: 保留盈餘合計。
- Equity: 權益總額。

## 損益表

API 路徑: https://api.finmindtrade.com/api/v4/data?dataset=TaiwanStockFinancialStatements&data_id={stock_id}&start_date={YYYY-MM-DD}

- date: 日期。
- stock_id: 股票代號。
- type: 數據類型，參考 `### 損益表數據類型` 的說明，表示此筆資料為該數據類型的數據。
- value: 值。

### 損益表數據類型

- Revenue: 營業收入。
- GrossProfit: 營業毛利。
- OperatingIncome: 營業利益。
- PreTaxIncome: 稅前淨利。
- IncomeAfterTaxes: 稅後淨利。
- TotalNonoperatingIncomeAndExpense: 營業外收入及支出。
- EPS: 基本每股盈餘。

## 現金流量表

API 路徑: https://api.finmindtrade.com/api/v4/data?dataset=TaiwanStockCashFlowsStatement&data_id={stock_id}&start_date={YYYY-MM-DD}

- Depreciation: 折舊費用。
- AmortizationExpense: 攤銷費用。
- PropertyAndPlantAndEquipment: 取得不動產、廠房及設備。
- ProceedsFromLongTermDebt: 舉借長期借款。
- RepaymentOfLongTermDebt: 償還長期借款。
- RedemptionOfBonds: 償還公司債。
- CashFlowsFromOperatingActivities: 營業活動現金。
- CashProvidedByInvestingActivities: 投資活動現金。
- CashFlowsProvidedFromFinancingActivities: 籌資活動現金。

## 股利政策

API 路徑: https://api.finmindtrade.com/api/v4/data?dataset=TaiwanStockDividend&data_id={stock_id}&start_date={YYYY-MM-DD}

- CashEarningsDistribution: 現金股利。
- StockEarningsDistribution: 股票股利。
- CashDividendPaymentDate: 除息日。
- StockExDividendTradingDate: 除權日。
