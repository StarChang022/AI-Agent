# Worklow 產品規格書

基於「極致去手動化」與「企業財務自動化起點」之願景，本規格書將 Worklow 的核心流程轉化為具體的系統架構、使用者動線與頁面功能。

## 1. 產品資訊架構 (Information Architecture)

Worklow 採 Web-based 架構為主，搭配輕量化的行動端介面（Web App 或 Native App）。
主要模組包含：
1. **儀表板 (Dashboard)**：即時預算與支出監控。
2. **進項收件匣 (Inbox)**：全通路單據匯入與狀態追蹤。
3. **審核中心 (Audit Center)**：雙窗格人機協作核帳。
4. **報表與導出 (Export)**：傳票生成與稅額抵減表。
5. **系統設定 (Settings)**：信任門檻與科目/部門管理。

## 2. 核心使用者動線 (User Flows)

### 動線 A：業務外勤「閃電報支」 (Mobile 導向)
目標：隨時隨地，零負擔記錄。
1. **開啟 App** 點擊「新增進項」。
2. **拍攝/匯入** 計程車收據或餐費電子發票。
3. **AI 瞬間辨識** 雲端處理並回傳結構化數據。
4. **快速確認** 檢視系統自動帶出的「金額」、「會計科目（如：交通費）」與「歸屬部門」。
5. **一鍵送出** 單據即刻進入雲端，完成報支申請。

### 動線 B：會計行政「月底批次核帳」 (Web 導向)
目標：處理大宗數據，從打字員轉為審核員。
1. **進入收件匣** 將數十封 PDF 電子發票或掃描檔拖曳至上傳區。
2. **系統初篩** AI 背景執行辨識與「信任門檻」邏輯校驗。
3. **批次放行** 系統列出 90% 無誤的「綠燈件」，會計一鍵全選並核准。
4. **例外處理** 點擊剩餘 10% 的「紅燈異常件」，進入審核中心修正（如：統編錯誤或數字模糊）。
5. **轉成傳票** 確認無誤後，一鍵生成會計傳票草稿。

## 3. 具體頁面與功能模組 (Key Pages & Features)

### 3.1 儀表板 (Dashboard)
管理層與財務長的「財務真相中心」，提供最即時的數據可視化。
- **預算執行監控卡片**：視覺化顯示各部門當月/當季預算達成率。支援紅綠燈號警示，預防超支。
- **部門費用對比圖表**：以直觀的圓餅圖/長條圖分析各部門支出結構。
- **待辦事項 (To-Do)**：提醒目前「待審核異常件」或「待轉傳票」的數量。

### 3.2 進項收件匣 (Inbox / Dropzone)
所有進項數據的統一入口，不論紙本或數位皆在此匯流。
- **多通道拖曳區**：支援上傳 PDF, JPG, PNG 格式。
- **狀態分類清單**：採用 Kanban（看板）或列表模式，分類為：「辨識中」、「待審核（異常）」、「待審核（正常）」、「已結案」。
- **批次處理進度條**：大量上傳時，提供清晰的 AI 處理進度指示。

### 3.3 互動式審核中心 (Audit Center)
專為財務人員設計的「虛擬審計助理」工作區，實現極速核帳。
- **雙窗格對比介面**：
  - **左側 (Source)**：顯示原始單據的高解析度影像，支援放大、縮小、旋轉。
  - **右側 (Data)**：顯示 AI 提取的結構化表單（發票號碼、日期、統編、金額等）。
- **紅字異常警示 (Exception Highlighting)**：對於信心值偏低，或是觸發數學邏輯不符（未稅+稅額≠總計）的欄位，強制標示紅框提醒。
- **自動會計分攤建議**：根據歷史報支習慣與員工屬性，右側表單自動帶出建議的「會計科目」與「部門歸屬」。
- **全鍵盤操作支援**：支援 Enter/Tab 等快捷鍵跳轉異常欄位與確認送出，達成「滑鼠零點擊」的極速體驗。

### 3.4 傳票與報表導出 (Reports & Exports)
打通財務自動化的最後一哩路。
- **傳票智慧建立**：將審核通過的進項，自動填入稅前與稅後金額，轉換為標準「會計傳票草稿」格式。
- **進項稅額抵減表**：自動挑出可扣抵的發票，生成清單，協助節稅與營業稅申報。
- **資料導出整合**：支援匯出為 CSV/Excel，或透過 API 推送至各大主流 ERP 系統（如 SAP, Oracle, 鼎新）。

### 3.5 系統設定 (Settings)
企業管理邏輯的控制台。
- **信任門檻機制設定**：管理者可設定 AI 信心值（例如 > 95%）與必須通過的邏輯校驗規則，達標者自動放行。
- **組織與科目維護**：同步或建立公司部門架構、預算上限、與會計科目對照表。
