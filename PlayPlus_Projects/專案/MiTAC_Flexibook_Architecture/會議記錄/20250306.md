## MeetingRoom UX 調整

1. 會議室清單調整。
    1. 基本篩選條件的 Site 右邊新增「區域」。
    2. 進階篩選條件與清單的條件 TAG 移除**關鍵字**。
    3. 近期預約要標示清楚，並且移動到其他位置，避免根 Filter 混雜。
    4. Hover 的X軸和Y軸加上顏色變化。
    5. 點擊 Table 的時段空檔會直接開啟 Modal 指定時間起訖。
    6. 時段符號放上提示說明。
2. 顯示上下午時段不影響搜尋結果。
3. 篩選抽屜調整。
    1. 樓層由後端決定，不一定每間 Site 都是「幾樓」。
    2. 人數選項調整為「0-10人 / 11-20人 / 20人以上」。
4. 篩選條件的**關鍵字**先拿掉。
5. 選擇預約時段 Modal 調整。
    1. 日期改為可以切換，切換的時候，起訖時間恢復空白。
    2. **重置起訖時間**按鈕要標明 Note 會讓起訖時間恢復空白。
    3. 點擊被預約的時段一樣要顯示預約資訊，但不要用 Modal 顯示。
    4. 起訖時段拖拉的頭尾要有顏色變化。
    5. **分機**與**手機號碼**位置對調。
    6. **同步使用 WebEx** 改為「不申請／WebEx／Teams」3選1。
6. 完成預約調整。
    1. **會議室使用注意事項**移動至下方。
    2. **WebEx 會議連結**根據**選擇預約時段 Modal** 所選擇的選項變化。
7. 會議室清單空檔卡片樣式的空檔時段標註點擊後不會有動作。
