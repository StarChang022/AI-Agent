## 一、私訊互動

### 功能描述

採購商可透過供應商頁面發起私訊對話，雙方可即時收發訊息。系統於訊息未讀超過 3 天後，自動通知管理者及未讀方。

### 已確認規格

| 項目 | 規格 |
| --- | --- |
| :pushpin: 對話對象 | *供應商 :left_right_arrow: 採購商*（採購商主動發起聯繫） |
| :pushpin: 發起方式 | 採購商到供應商頁面發起聯繫，*無需先有詢價關係* |
| :pushpin: 即時推播 | 需支援 WebSocket 即時推播（ActionCable） |
| :pushpin: 未讀提醒 | *3 天*未讀未回覆後觸發通知 |
| :pushpin: 提醒頻率 | *只提醒一次*，之後不再重複通知 |
| :pushpin: 通知對象 | 通知*管理者*（純通知，無需介入）及*未讀取訊息的人* |
| :pushpin: 附件支援 | 支援*圖片（jpg/png）+ PDF*，單檔上限 *5MB* |

## 二、新品快訊

### 功能描述

定期將近期審核通過的新產品，透過 Email 通知方案有效的採購商。

### 已確認規格

| 項目 | 規格 |
| --- | --- |
| :pushpin: 通知對象 | 僅限*方案有效（付費中）的採購商* |
| :pushpin: 通知頻率 | 每 *7 天*（後台可調整） |
| :pushpin: 退訂機制 | *不可退訂*，強制接收 |
| :pushpin: 新上架定義 | 以*審核通過日期（permitted_at）*為準 |
| :pushpin: 天數設定 | 與「首頁新品專區」*共用同一個後台設定* |

## 四、5 星 Plus 專區

### 功能描述

5 星供應商的「廠區衛生」區塊顯示「+」符號，點擊展開可下載的附件檔案。由後台管理員上傳，需經管理員審核開啟。

### 已確認規格

| 項目 | 規格 |
| --- | --- |
| :pushpin: 檔案綁定層級 | 綁在*供應商（SupplyEnterprise）層級* |
| :pushpin: 瀏覽權限 | 綁在*廠區衛生分數申請*：採購商需申請過任一間廠區的衛生分數後，才能看到分數及 Plus 檔案 |
| :pushpin: 下載權限 | *看得到就能下載*（申請過衛生分數的人都能下載） |
| :pushpin: 上傳者 | *後台管理員*手動上傳 |
| :pushpin: 檔案格式 | *僅 PDF* |
| :pushpin: 開啟方式 | 達 5 星後，需*管理員額外審核*才能開啟 Plus 功能 |
| :pushpin: 降星處理 | *隱藏*（不刪除），恢復 5 星後自動重新顯示 |
| :wrench: 檔案大小上限 | 開發時再決定 |

## 五、PDF 浮水印

### 功能描述

指定類型的 PDF 檔案下載時加上固定文字浮水印。

### 已確認規格

| 項目 | 規格 |
| --- | --- |
| :pushpin: 處理範圍 | 僅 PDF 檔案，Word / 圖片不處理 |
| :pushpin: 浮水印類型 | *固定文字*（效能最佳，可預處理） |
| :wrench: 浮水印文字 | 開發時再決定（如平台名稱） |
| :wrench: 浮水印樣式 | 位置、透明度開發時再決定 |

:pushpin: *需加浮水印的檔案（客戶指定）*：

| # | 檔案類型 | 來源 |
| --- | --- | --- |
| 1 | 產品責任險 | SupplyEnterprise 供應商資料上傳 |
| 2 | 產品合規附件（供應商資料內） | InspectionReport.certified_file |
| 3 | 管理制度附件（場域列表下方） | ManagementSystem.certified_file |
| 4 | 產品合規附件（產品管理下方） | 產品相關的合規文件 |
| 5 | 實地查證表 | FactoryCheckForm → wicked_pdf 動態產生 |
| 6 | 5 星 Plus 附件 | 後台上傳的 5 星 Plus 檔案 |

:pushpin: *不需要加浮水印的檔案*：

- 使用者同意書
- 平台使用指南
- 使用同意書範本
- 服務相關附件

## 七、數據報表

### 功能描述

後台新增「數據報表」頁面，支援日期區間篩選，含多個子報表，全部可匯出 Excel。

### 已確認規格

| 項目 | 規格 |
| --- | --- |
| :pushpin: 報表匯出 | 全部支援 *Excel 匯出* |
| :pushpin: 文件下載追蹤 | *累計總數*即可（不需逐筆紀錄下載者） |
| :pushpin: 產品數量定義 | 該期間內狀態為*已審核（permitted）的產品總數* |
| :pushpin: 漏斗呈現方式 | *表格 + 圖表都要*（圓餅圖或長條圖） |

### 子報表規格

### 7-1. 會員註冊

篩選日期區間內，*通過註冊審核*的新會員數量。

### 7-2. 有效會員

:pushpin: *計算方式*：篩選區間內的*每日有效會員數量*。

- 針對篩選區間內的每一天，計算該日方案仍有效的會員數
- 呈現方式：日期 × 有效會員數的表格 + 折線圖

### 7-3. 會員方案歷程

顯示會員每次啟用的方案名稱及起迄日。可依會員搜尋。

### 7-4. 會員方案漏斗

顯示篩選期間內各方案的分佈數量及百分比（A / B / C 方案各佔多少）。
呈現方式：*數據表格 + 圖表*。

### 7-5. 產品數量

篩選期間內，狀態為 permitted 的產品總數。

### 7-6. 文件下載次數

每個文件的累計被下載次數。
