### 一、 網站核心定位（主要內容）
「中保無限+ 生活誌」是 **中興保全科技（SECOM）** 旗下的官方內容媒體平台。它打破了傳統保全產業冷冰冰、純技術的形象，將「智慧科技」與「日常生活」深度結合。網站主要涵蓋 **AI 科技應用、居家安全、健康照護、產業趨勢、建築美學及美食享樂** 等多元內容。

其核心目標在於透過高質感的內容產出，建立品牌專業度與信任感，並潛移默化地推廣中保科技集團的多元服務（如 AED 佈建、智慧建築、餐飲品牌等）。

### 二、 頁面架構與任務分工
網站採用直觀的頂部導覽與模組化首頁設計，主要頁面及其任務如下：

| 頁面類型 | 頁面任務 |
| | |
| **首頁 (Home)** | **品牌櫥窗：** 整合最新話題、熱門影片及特企專題，引導使用者快速進入感興趣的分類。 |
| **分類頻道頁** | **深度導覽：** 如「智慧科技」、「美好家居」等，針對特定受眾提供垂直領域的資訊，強化專家形象。 |
| **特企專題頁** | **沉浸式行銷：** 針對大型專案（如國產建材、ESG 永續）進行整合性報導，提升品牌深度。 |
| **文章內頁** | **資訊傳遞與轉化：** 提供清晰的閱讀環境，並透過文末推薦連結或社群分享按鈕，增加使用者黏著度。 |
| **影音特區** | **動態參與：** 利用 YouTube 嵌入式內容，以更具感染力的影像傳達品牌精神。 |

### 三、 網站關鍵功能
為了提升使用者經驗與互動性，網站配置了以下功能：

* **智慧搜尋系統：** 提供熱門關鍵字推薦（如：智慧城市展、行動 AED），縮短資訊尋找時間。
* **多樣化媒體嵌入：** 支援高畫質影片播放與 Facebook 粉絲專頁即時動態串接。
* **社群互動分享：** 每篇文章皆整合 LINE、FB、Twitter 等一鍵分享功能，提升內容擴散率。
* **響應式設計 (RWD)：** 優化移動端閱讀體驗，適應手機、平板等不同裝置。
* **廣告與專案看板：** 側邊欄與首頁區塊可彈性置入重點專案或集團最新快訊。

### 四、 視覺調性 (Visual Tone)
本網站在視覺設計上追求「專業信任」與「溫暖生活」的平衡：

* **色彩配置：** 以中保品牌指標性的 **鮮紅色** 作為重點強調色（點綴於標題與圖示），搭配大量 **純白空間**，營造現代、乾淨且權威的閱讀感。
* **版型設計：** 採用 **雜誌式佈局 (Magazine Layout)**，利用卡片式設計排列圖文，使資訊分塊明確，具備高度的掃視性 (Scannability)。
* **圖像風格：** 使用大量高品質的實境攝影照片，強調「人」與「空間」的互動，降低科技的距離感，增加品牌親和力。
* **字體運用：** 採用清晰的無襯線字體，確保在長篇文章閱讀時的眼力舒適度。

### 結語
這不僅是一個資訊發布網站，更是中興保全科技與消費者溝通的橋樑。它將複雜的 **AIoT 解決方案** 轉化為易懂的 **生活故事**，是企業數位轉型與內容行銷的典範之作。
