# 《ROE》與《ROA》評估企業賺錢效率與品質，看看你的投資有沒有被高效運用？

Warren Buffett 曾說「我最喜歡的持股時間是永遠」，他將「買股票」這件事視為真的買進公司，不僅僅是買進帳面上的股數，而買進公司就必須先考慮該公司的賺錢效率。

![image.jpg](attachment:19174bcb-b799-45f8-b14d-c412b17f3d90:image.jpg)

想像一下，如果你要買進一家公司、買進一樁生意，你會最關心什麼？我猜你會最關心「賺錢效率」對吧！買進公司就是要賺錢呀，所以你會在意每一塊錢的投入能換回多少錢。

財務報表裡面就有個「股東權益報酬率（ROE）」指標用來評估賺錢效率，讓我們像 Warren Buffett 一樣思考如何用 ROE 來找出值得長期持有的好公司。

## 為何 Warren Buffett 看重 ROE？表示企業的賺錢效率

<aside>

ROE = 稅後淨利 / 股東權益 x 100%

</aside>

你常聽到的「股東權益報酬率（ROE）」是衡量公司賺錢效率的核心指標，概念是「公司用股東的錢賺回多少利潤」。

公式裡的「股東權益」等同於公司的「淨值」，也就是總資產扣除所有負債後，真正屬於股東們的自有資本。而前面損益表篇章有提過，「稅後淨利」是扣除稅金後真正獲得的利潤。所以 ROE 公式就是在說自有資本賺到利潤的倍數，通常為每季或每年為時間單位，倍數越高則賺錢效率越好，表示每1塊錢能賺到多少倍利潤。

這就是為什麼 Warren Buffett 如此重視這個指標，一家長期維持高 ROE 的公司，代表它擁有非常強大的獲利引擎，不需要一直向股東伸手要錢（辦理現金增資），就能靠著自己賺來的利潤持續成長，這正是創造驚人複利的關鍵，這就是一門能自己滾雪球的好生意。

## ROE 與 ROA 為何要搭配使用？從多個角度檢視獲利

我們現在知道 ROE 如同一具強大的獲利引擎，但如果開車只看時速表，卻忽略了油箱還剩多少油，那是非常危險的。單獨看 ROE 這個指標，有時會讓我們忽略獲利的「品質」，以及公司潛在的風險。

### ROE 的陷阱：你看到的是真實力，還是債務堆出的海市蜃樓？

剛剛提到的公式「ROE = 稅後淨利 / 股東權益」乍看之下很合理，其實單獨看這項指標會有盲點，假設今天有企業向銀行貸款一大筆錢，用來買地建廠、大幅擴張，不管效率如何，營收和利潤數字一定會提高，但股東權益不會變，所以 ROE 就會跟著提升。又或者，大幅增加存貨來換營收，同樣會提高利潤數字，但股東權益不會變。

這就像一位吃禁藥的短跑選手，成績驚人，但我們都知道這不是真實力，而且潛藏著巨大的風險，一旦情勢逆轉，例如升息導致還款壓力變大，這位選手就可能立刻崩潰。

### ROA 的角色：用更全面的角度檢視公司體質

<aside>

ROA = 稅後淨利 / 平均總資產 × 100%

</aside>

剛剛的 ROE 是討論「淨資產」的賺錢效率，ROA 則是從另一個角度討論「總資產」的賺錢效率。

總資產是連同負債都算入的數字，把公司自有資金和借來的錢全部都納入考量，所以公司利用負債槓桿或者增加存貨來提升營收數字，躲不過 ROA 的檢視。

1. 高 ROE、低 ROA： 這家公司的獲利，很可能大部分是靠借錢撐起來的。
2. 高 ROE、高 ROA： 這是真功夫！光靠本業經營就能為股東創造優異的回報。

## 長期觀察 ROE 與 ROA 看見企業的獲利能力護城河

我們已經學會如何運用 ROE 和 ROA 來評估一家公司的賺錢效率與獲利品質。而巴菲特的投資哲學中，重視優質且長期穩定的 ROE 與 ROA，這相當於某種護城河。

任何公司都可能因為一時的好運、搭上產業熱潮，或是一次成功產品，在一兩年內交出漂亮的財報，但是曇花一現的榮景，很快就會被無情的競爭所侵蝕。

只有真正偉大的公司，才能夠抵禦這些競爭者的攻擊，所以能夠長期維持賺錢效率，一定是本身具備某種護城河，例如品牌價值（如可口可樂）、低成本結構（如好市多）、高轉換成本（如蘋果生態系）、社群效應（如 Facebook）。

## 我們要挑會賺錢且健康的企業

看完 ROE 和 ROA 的說明，希望能讓你對如何評估一家公司有不同角度可以切入。讓我們記住，一門好生意不僅要很會為股東賺錢（高 ROE），更要賺得聰明、賺得健康（穩健的 ROA），而這一切的背後，必然有一條強大的護城河在默默守護。
