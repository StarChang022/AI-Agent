# 《流動比率》與《速動比率》是什麼？類似企業償債能力的「財務急救箱」

為什麼有些公司看起來資產很多，市場崩盤時卻還不出錢來？關鍵在於資產的『變現能力』。

打開企業的『財務急救箱』，快速判斷一家公司能不能撐過危機。透過『流動比率』與『速動比率』這兩個關鍵指標，五分鐘內學會分辨誰是強壯的倖存者，誰又是外強中乾的地雷股。

![current-ratio-and-quick-ratio.jpg](attachment:e7e7910a-6820-4699-80a4-2a90ec68c28e:current-ratio-and-quick-ratio.jpg)

作為一個謹慎的投資者，你最害怕的，或許不是股價的短期漲跌，而是你辛苦存下的血汗錢，會因為公司突然倒閉而一夕歸零。

市場崩盤時，體質脆弱的公司就像紙牌屋一樣不堪一擊，如何辨識財務結構脆弱的地雷公司？翻開公司的「財務急救箱」。急救箱的內容物就是流動比率（Current Ratio）與速動比率（Quick Ratio）。從這兩個指標來看，五分鐘內就能判斷一家公司面對危機時的生存能力。

## 什麼是流動比率與速動比率？類似公司財務的急救箱

前言我們提到「財務急救箱」概念，用來判斷一家公司的財務體質。現在，我們打開這個急救箱，看看裡面放了些什麼。

急救箱有兩層，下層是我們平常準備周全的流動比率「完整急救箱」；上層則是可以隨手拿起來應對突發狀況的速動比率「緊急創傷包」。

### 流動比率：完整急救箱

<aside>

流動比率 = 流動資產 / 流動負債

</aside>

想像一下，流動比率就類似你家中那個最完整的急救箱，實際意義是代表公司「一年內能變現的資產」來償還「一年內該付的債務」的能力。簡單來說，就是回答投資者「所有短期資源夠不夠拿來還債」。

流動資產包含**現金和約當現金**、**銀行存款**、**有價證券**、**短期投資**、**應收帳款**、**應收票據**、**存貨**、**預付費用**。流動負債則是指公司在一年內必須償還的短期債務，例如短期借款或應付帳款。

### 速動比率：緊急創傷包

<aside>

速動比率 = ( 流動資產 - 存貨 - 預付款項 ) / 流動負債

</aside>

假設意外發生一個更危急的狀況，需要立刻止血，這時你應該很難慢條斯理地翻找整個急救箱，而是會立刻抓起身邊的「緊急創傷包」。速動比率是一個更嚴格、更保守的生存指標，代表對企業應急能力的嚴苛的考驗。

為什麼要特別把「存貨」拿掉？因為存貨是所有流動資產中，變現能力最差、不確定性最高的一項。尤其是有過季問題或期限問題的產品，例如舊款手機庫存可能在一年後就一文不值、食品庫存則可能面臨腐壞的風險。

至於「預付款項」則表示公司先支付款項，但尚未得到商品或服務（例如預付租金或保險費），這些已經付出去的錢，短期幾乎不具備償債能力。

因此，速動比率往往更能真實反映一家公司在緊要關頭，能立即拿出多少資源來還債。

## 流動比率多少算是合理？

通常，市場上有普遍認知的「標準答案」。

- 流動比率 > 200%
- 速動比率 > 100%

這個標準背後的邏輯很直觀，流動比率超過 200% 代表公司擁有的短期資產，是短期負債的兩倍以上，提供一個舒適的安全緩衝區。

而速動比率超過 100% 則意味著即使完全不依賴存貨，公司也能立即清償所有短期債務，這無疑是一個令人安心的信號。對於所有人來說，都是一個可以快速篩選掉體質明顯虛弱公司的濾網。

## 償債能力

分析一家公司的安全性，不是只有分析「負債」的部分，償債能力同樣重要，就像文中一直使用的「急救箱」比喻。

每家公司的流動比率和速動比率普遍還是會有所不同，尤其是跨產業的時候。當你發現異樣時，可以深入研究是哪邊造成差異，同業是不是也都差不多。
