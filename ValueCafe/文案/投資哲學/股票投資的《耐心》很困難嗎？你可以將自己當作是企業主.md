# 股票投資的《耐心》很困難嗎？你可以將自己當作是企業主

種下一顆種子，你不會期望明天就看到它成長為大樹；經營小攤車，你不會期望明天就變成連鎖店。因為你知道需要時間、需要耐心，直到你獲得豐碩成果才發現，每天默默做點什麼已經成為習慣。

投資完全是同樣道理，你當初選擇一家被低估的優秀企業，買進後就抱著，你不要期望明天就能致富，它會慢慢為你帶來回報。你需要做的只是定期檢查有沒有什麼新消息、最新一季財報表現是否仍在軌道上。

![Patience.jpg](attachment:cbc9cdcc-8bad-4fa9-a076-981a7b7f6e77:Patience.jpg)

1986年 Warren Buffett 做了一項他坦承一竅不通的投資，買下一座位於奧馬哈的農場 。他不懂農耕，但他兒子熱愛務農，告訴他這個農場可以種出多少玉米和大豆，以及經營費用大概是多少。

他算了一筆簡單的帳，根據這些估計，計算出這個農場的正常報酬率約為10％。長期而言，農場的生產力將成長，而農產品的價格也將上升。買下之後，再也沒想過要為這塊土地找個每日報價，或者為隔壁農地價格的漲跌而感到焦慮。

當然，農場收成有時候會表現差勁，但有什麼關係？只要長期持有，農場也會有非常好的時候，而且我們沒有出售這項資產的壓力。

話題拉回來，為什麼心平氣和地持有一座農場數十年，似乎是理所當然的事，卻無法用同樣的心態，持有手中股票？

投資的世界裡，多數人都活得像個焦慮的「股價租客」，而不是老神在在的「企業主人」。我們關心螢幕上閃爍的價格，遠勝於關心企業本身的價值。這正是建立穩固價值投資心態的第一道障礙，也是投資心態建立中最常被忽略的一環。

## 為何耐心如此困難？

當我們將自己定位為「股價租客」時，等於是把情緒的主導權，完全交給市場的喜怒無常。

你會發現自己每天都在猜測市場的心情，今天市場是漲是跌？新聞說了什麼？專家又預測了什麼？在這種心態下，當股票下跌時，恐懼與恐慌是必然的反應。

但，還有另一種選擇。

那就是選擇成為「企業主人」，就像是剛剛比喻的耐心的農場主，讓我們來看看這兩種心態在面對同一個市場時，會產生哪些不同的反應與結果。

| 角色 | 焦躁的交易員 (股票租客) | 耐心的農場主 (企業所有者) |
| --- | --- | --- |
| 核心焦點 | 每日的市場價格波動。 | 企業的長期生產力與盈利能力。 |
| 時間視野 | 分鐘、天、或月。 | 年、數十年、或一生。 |
| 資訊來源 | 財經新聞、專家預測、市場「噪音」。 | 公司年報、產業分析、商業基本面。 |
| 焦慮來源 | 「我能準確預測市場時機嗎？」 (FOMO與恐懼) | 「這家企業的根本生產力是否依然強勁？」 |
| 對市場崩盤的看法 | 一場需要立即行動（賣出！）的恐怖危機。 | 一個「歉收年」，或是以低價購買更多「土地」的機會。 |
| 「行動」的定義 | 頻繁的買入和賣出。 | 最初的購買決策，以及隨後刻意的、耐心的持有。 |
| 終極目標 | 智勝市場以獲取短期利潤。 | 與一家偉大的企業合作，讓財富隨時間複利增長。 |

如果你發現自己總是在左邊掙扎，這份焦慮並不是你的錯，純粹是觀點的選擇。你不需要更強的意志力，你需要的是一套全新的作業系統。

## 建立長期持有的企業主觀念

從短線心態移動到長期心態，不需要天賦，只需要將觀念刻印在心裡。

Warren Buffett 的農場故事，不僅讓他自己獲得啟發，也為我們提供很好的參考，清楚理解長期持有的「企業主」觀念會是什麼樣子。

### 「每日報價」焦點轉移為「每年產出」

交易員每天早上醒來，優先關注「今天的股票報價是多少」，而農場主偶爾會思考「今年的收成會如何」，這個微小的差異，決定了截然不同的情緒與結果。

Warren Buffett 雖然對經營農場一竅不通，但估算出這塊土地每年預期約 10% 的年化回報率。他的決策依據，是資產的盈利能力，也就是這塊「土地」能為他生出多少錢，而不是資產的轉售價格。

畫面轉移至股票市場，它為你的農場提供每秒鐘都在變動的報價，這也正是所有投資焦慮的根源；農場主之所以能保持平靜，正是因為沒有人會每天來為他的土地報價，他可以專注於真正重要的事情「土地的生產力」。

這就是長期價值投資的關鍵，你的焦點應該放在企業的年度財報上，而不是券商 APP 的報價。

### 「市場崩盤」風險重估為「歉收之年」

交易員看待風險的方式是「價格下跌」，而農場主看待風險的方式是「產量減少」。前者是恐慌的來源，後者則是經營的常態。

投資過程中必然會遇到「偶爾的壞收成」或「令人失望的價格」，一場 50% 的市場崩盤，在交易員眼中是末日降臨，但在農場主看來，這只是其中一個「歉收年」。

只要農場的土地依然肥沃（企業的基本面依然強勁），一個歉收年並不可怕。你要做的不是在恐慌中把土地賤賣掉，而是耐心等待下一個豐收季節的到來。甚至，如果鄰居的農地因為歉收而恐慌拋售，你還可以用便宜的價格買入更多肥沃的土地。

### 「頻繁交易」行動改變為「不作為」

對交易員來說，買進和賣出是每天必要的行動；對農場主來說，最重要的行動，就只是最初那個正確的購買決策，後續則是長達數十年的「不作為」。

Warren Buffett 提到，持有農場到現在，他只去過兩次。巨額利潤幾乎全部來自於1986年的那個購買決策，以及隨後漫長歲月裡的耐心等待。真正的投資收益，來自於你『不做什麼』的長期複利。

當你完成深入研究，買入偉大企業的部分所有權，你就已經完成了99%最重要的工作。剩下的1%，就是戰勝內心的衝動，讓企業的價值隨著時間的雪球慢慢滾大。

## 當個企業主，不要只盯著價差

當你真正將自己看作企業主或農場主，而非一個時時緊盯報價的租客時，你會感受到一個深刻的轉變。你關心不再是短期的價差，而是長期的價值。

真正的投資收益，來自於你「不需要做什麼」的決定。

當你選對企業，剩下最重要的工作，就是給予時間及耐心，讓價值慢慢發酵，最終會收穫豐厚的回報。還有，免於焦慮的自由。
