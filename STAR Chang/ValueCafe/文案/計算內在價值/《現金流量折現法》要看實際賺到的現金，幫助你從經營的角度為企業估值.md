# 《現金流量折現法》要看實際賺到的現金，幫助你從經營的角度為企業估值

「現金殖利率估值法」很有意思，這套算式源自於 Warren Buffett 對現金的熱愛，他認為企業的價值有很大程度由「能夠賺進多少現金」來決定，賺到現金才是實實在在的企業盈利。

當然不能只是看看今年，更要看未來五年十年，於是 Warren Buffett 創造這麼一套公式，估算企業未來能夠賺多少錢，扣掉通貨膨脹後再回推至今年計算現值。如此得到，我們能夠買到多少未來價值。

![discounted-cash-flow.jpg](attachment:4d5dcb1c-7b6b-4af6-96a9-c765ee7826e7:discounted-cash-flow.jpg)

你有沒有思考過，股價漲跌到底是在反應什麼？

每天的財經新聞、分析師報告、令人眼花撩亂的本益比（P/E）和股價淨值比（P/B），被這些資訊轟炸後感覺好像懂了些什麼，深入探究才發現什麼都不懂。

我們都知道要「逢低買進」，但何時是「低」？我們也都知道要「避免追高」，但股價的「高」在哪？關掉所有噪音、回歸原點，忘掉那些複雜的指標和圖表，我們從本質探討，一家公司真正的價值該如何衡量。

做生意、開公司、經營企業的本質就是「賺錢」嘛！更精確地說是「賺現金」，所以當我們為公司計算估值，從「未來能夠賺多少現金」的角度來看，就是從本質探討真正的價值，也就是「現金流量折現法」，簡稱 DCF。

## 1. 《自由現金流》這生意每年能為我帶來多少現金？

假設你是位企業家，當你走進一間你想要收購的咖啡店，你評估價值時想釐清的問題，應該不會是「本益比多少」吧？而是最根本的問題「扣掉所有開銷後，咖啡店每年到底能為我帶來多少現金」。

老闆拿出財務報表，指著「淨利潤」說「看，我們去年賺了50萬，成績不錯吧」，數字看起來很吸引人，但身為精明的買家，你沒有馬上點頭，你深知，「帳面上的利潤和真正放進口袋的錢」是兩回事。

### 為什麼不看「淨利潤」，要看「自由現金流」？

「淨利潤」是一個會計概念，有時候會騙人（或者說誤導），而「自由現金流」則是企業真正產生的現金。

此前曾經說明過自由現金流，從損益表換算過來會有「消失的現金」和「多出來的現金」，也就是說兩者之間會有一些差距（所以有心人士經常在損益表進行誤導）。

1. 消失的現金（資本支出）： 義式咖啡機可能明年就得花20萬買1台新的，這筆20萬的「資本支出」不會在損益表裡面被扣掉（會用折舊的方式）。然而，這卻是真實從你口袋裡流出的現金。
2. 多出來的現金（折舊）： 同時，咖啡店去年的利潤50萬，包含會計師扣除一筆5萬元的「設備折舊」費用，這是三年前買的舊咖啡機的成本攤銷，但這筆錢並沒有真正從你口袋裡掏出現金。

站在「企業家」的角度來看，你應該要計算的是「真正賺進口袋的錢」而非「帳面上的業績」（業績是交代給投資人看的）。「現金流量折現法」是 Warren Buffett 站在企業家的角度思考所得出的估價邏輯，因此必須將「自由現金流」作為評估的起點。

## 2. 《折現率》明年的100萬跟今天的100萬相同價值嗎？

透過檢視帳本與未來的規劃，你大致推算出這家咖啡店在未來幾年能帶來的自由現金流。假設你預估明年它能產生100萬的現金，準備掏錢收購之前，還得再想想「明年的100萬，對今天的我來說，真的值100萬嗎？」

「貨幣的時間價值」是財務世界最基本的常識，今天的錢，永遠比未來的錢更有價值。而「折現」就是把未來的錢，換算成今天價值的過程，這個換算的比例，就是所謂的「折現率」。

就像你看有些節目或歷史資料，在談那個年代的富人財產，換算成今天可能價值多少美元，這也是折現換算。

### 所謂「折現率」就是指你對這筆投資的最低要求

先說「折現率」沒有標準答案，當年 Warren Buffett 將美國30年期公債殖利率作為折現率。意思是說，手上資金不一定非得用來投資股票或我們正在關注的標的，我們也可以選擇買入美國30年期公債這種毫無風險、每年可以穩賺 8%（假設是 8%）的標的，這就是「機會成本」。

回頭看看這家咖啡店，收購後需要投入時間、精力，還要面對各種風險，如果你下定決心走這條艱難道路，至少要賺得比機會成本的 8% 更多，這是必須的吧！因此你就可以抓一個高於 8% 的報酬率數字，當作你的最低期望報酬率，例如 13% 報酬率，這也就會是你設定的「折現率」，後續計算就都使用 13% 折現率。

折現率設定多少沒有標準答案，你可以根據自身情況，對這筆投資設定自己「最低期望報酬」。

## 3. 《終值》五年之後或更久之後呢？如何評估一家店的永續價值？

你已經預估出咖啡店未來五年的自由現金流、也決定要求 13% 折現率。再來還有個問題，一家值得收購的好店，理論上可以經營超過五年，甚至是幾十年。但是要精準預測一家公司十年、二十年後的現金流，這是辦不到的。

所以我們需要評估一個適用於「從第六年開始直到永遠」的未來價值。你要記得，這只是假設性的模型公式，但我們還是需要這個假設，才有辦法評估。

華爾街對此發明了一個詞「終值」，可以想像成是你在經營五年後，準備將這家成熟穩定的咖啡店賣掉時，能夠得到的合理價格。

不需要猜測第六年、第七年 ... 第十九年的營收，概念上比較像是「當這家咖啡店的營運模式步入穩定軌道」，第五年的年底，市場上另一位買家會願意用第五年自由現金流的幾倍價格來收購。這個思路其實很常見，評估房地產或收購其他小生意時，人們也常用年租金或年利潤的倍數來估算價值。

終值是我們 DCF 公式裡的最後一塊拼圖，預估一家好公司在可預測期間的所有潛在價值。所謂的可預測期間，我們通常會抓20年。

## 現金流量折現法的完整算式

我們剛剛試著思考三個商業的**公式模型**。

1. 這生意未來能賺多少現金？預估未來自由現金流。
2. 未來的錢，今天值多少？決定個人的折現率。
3. 更長遠的價值如何估算？估計一個合理的終值。

### 現金流量折現法公式模型

![截圖 2025-08-30 上午11.54.24.png](attachment:1821e4fe-465d-49dc-921d-ccc122910ad8:截圖_2025-08-30_上午11.54.24.png)

- CF = 你預估的未來各期「自由現金流」。例如第1年50萬、第2年55萬、第3年60萬、第4年65萬、第5年70萬，通常我們自由現金流會抓第1年至第5年，第6年開始則帶入終值。
- r = 你決定的個人「折現率」。例如剛剛提到的 13%。
- TV = 你估算的「終值」。例如你認為第5年能夠以當年底的自由現金流70萬的10倍將咖啡店賣出，那你就帶入700萬。

### 實際計算

公式模型看起來很複雜，別怕，其實很簡單。回頭看看三個**公式模型**，都換算成具體數字後，就可以帶入公式進行計算。

**1. 先計算前五年的現值**

- 第一年現值: 50÷(1+0.13)¹ = 44.25 萬
- 第二年現值: 55÷(1+0.13)² = 43.08 萬
- 第三年現值: 60÷(1+0.13)³ = 41.58 萬
- 第四年現值: 65÷(1+0.13)⁴ = 39.86 萬
- 第五年現值: 70÷(1+0.13)⁵ = 37.99 萬

**2. 再計算折現終值**

- 終值的現值: 700÷(1+0.13)⁵ = 379.94 萬

**3. 全部加總**

咖啡店的內在價值 = 44.25 + 43.08 + 41.58 + 39.86 + 37.99 + 379.94 = 586.7 萬

根據你的預估與要求，這家咖啡店的「內在價值」大約 587 萬，就是你的購買價格參考依據。放在股市上也是同樣道理，當你用現金流量折現法計算出內在價值後，你就可以將該金額作為買賣的參考依據。

## 現金流看估值

請記住！「現金流量折現法」模型（其實任何估值模型都一樣）因為是預估未來，所以裡面的參數全部都是假設值，沒有 100% 準確這種事情。只要任何一個假設稍有改變，這個數字就會跟著變動。

其實計算價值本身最有價值的地方是「過程」，因為它迫使你思考整個企業的商業面。當股價發生狀況時，你可以冷靜地自問「只是股價發生變化」還是「長期賺錢能力衰退」。

啊還有，「現金流量折現法」適合用於計算業務**相對穩定、可預測**的公司的內在價值，因為需要推算未來N年的現金流。
