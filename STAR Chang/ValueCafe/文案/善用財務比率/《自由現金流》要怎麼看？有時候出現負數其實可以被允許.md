# 《自由現金流》要怎麼看？有時候出現負數其實可以被允許

自由現金流出現負數，代表公司快倒閉了嗎？負數有分『好的』與『壞的』。好的負數可能是崛起的關鍵，壞的負數則是倒閉的前兆。我們透過一張表判斷公司是在『投資未來』還是『失血求生』。

![free-cash-flow.jpg](attachment:aef9def3-7908-4cf3-a5cb-4b22d3b3f883:free-cash-flow.jpg)

想像一下，週六夜晚，餐廳座無虛席，當晚結帳時，收銀機滿滿的鈔票（順便回顧一下這叫做營業收入），生意非常火熱，原來這是你開的餐廳，讚哦！

差不多一整個月都維持在火熱狀態，終於撐到月底，可以開心結算本月利潤。頂級食材、餐廳房租、員工薪水和添購設備等費用，加加減減算下來，竟然發現「淨利潤」和「剩下的現金」對不上？

現實世界的財報確實是如此，即便沒有花掉你一毛錢現金，很多「費用」也會從利潤中被扣除（例如設備折舊）；或者你明明沒賺到現金，帳面上的營收卻非常漂亮（例如庫存灌水）。很多看起來很賺錢的公司，實際上可能現金流非常緊張，所以 Warren Buffett 非常重視「自由現金流」，這份報表在財報上是騙不了人的。

## 為什麼財報上的「自由現金流」比「淨利」更誠實？

損益表上的「淨利」，就像一張精心化妝、修過圖的精美照片，公司展示給你看到的最美好的樣貌，會計師可以用合法的規則，讓數字看起來更漂亮。

至於現金流量表上的「自由現金流」，則像一份未經修飾的健康檢查報告，你只能看到真實的樣貌，現金流入就是流入、現金流出就是流出，幾乎無法說謊。

### 「自由現金流」與「淨利」的真實比較

下方為簡易的損益表和現金流量表，我把幾個關鍵項目寫進去作為範例。

透過簡易表格，你可以看到兩邊有關聯，但結算數字不相同。有些項目在損益表是減項，到了現金流量表反而要加回來，這是因為邏輯不同。「損益表」負責計算業績、「現金流量表」負責計算實際收到多少錢，價值投資者會從中尋找「既有成長動能」且「有真正收到錢」的公司。

**簡易損益表**

| 營業收入 | $1,000,000 |
| --- | --- |
| 毛利 | $700,000 |
| 營業利益 | $300,000 |
| 淨利 | $250,000 |
- **營業收入**： 這段期間賺到的業績，但其中有部分是庫存，尚未真正賣出。
- **毛利**： 扣掉食材成本，計算出毛利。
- **營業利益**： 扣掉租金、員工薪資和設備折舊，計算出營業利益。
- **淨利**： 再扣掉業外收支，最後得到的淨利數字。

**簡易現金流量表**

| 淨利 | $250,000 |
| --- | --- |
| 設備折舊 | +$80,000 |
| 庫存 | -$100,000 |
| 添購設備 | -$65,000 |
| ▶️ 自由現金流 | $165,000 |
- **淨利**： 營運從損益表的淨利當作現金流量表的計算起始。
- **設備折舊**： 損益表上發生的設備折舊，只是根據會計規則需要作帳攤提，實際上設備款項並不是在這段財報期間支付，而是以前就已經結清，所以在現金流量表這邊要加回來。
- **庫存**： 對於損益表來說，增加庫存一樣會算在營業收入內，但庫存表示尚未真正賣出賺到現金，所以在現金流量表這邊要扣掉。
- **添購設備**： 添購設備這件事情對損益表來說，會是用「分批折舊」的形式慢慢扣減，對於現金流量表來說則是「一次支出」。所以這邊的計算就會和損益表有一段落差出現。
- **自由現金流**： 從第一項的**淨利**逐項加減後所得到的結果。

## 自由現金流是負數，代表公司快倒了嗎？

其實還滿像膽固醇的，超標乍聽之下很嚇人，實際上可以區分為「好的膽固醇」和「壞的膽固醇」。當你看到一家公司的自由現金流是負數時，要區分是「好的負數」或「壞的負數」，學會像一位資深分析師，解讀負數的根源。

### 好的負數是為了投資未來

「好的負數」通常是一筆巨大、著眼於未來的策略性投資。本業很賺錢，能源源不絕地吸進現金，但管理層決定將這些賺來的錢再加上一部分儲備資金，以及銀行借貸，全部投入到一個能開創更多營收的項目上（例如廠房或生產線）。這時候會看到支出的總金額大於收入的總金額，於是自由現金流變成負數。

這個負數嚇人嗎？不會，這不是危機，而是雄心的證明，這代表公司正在為明天更巨大的成功播種，投資未來。 早年的 Amazon 就是最經典的例子，它長期將賺來的現金全部投入建設倉儲和研發，導致自由現金流經常是負數，但它也因此打造了無人能敵的電商帝國。

不過，要是連續好幾個會計期間（好幾季或好幾年），都因為投資未來而呈現負數，那也得注意是不是過度下賭注。

### 壞的負數是正在失血的警訊

「壞的負數」則危險得多，根源不在於投資未來，而是連核心業務本身都無法產生正向現金流。這意味著，公司光是維持日常營運（賣產品支付開銷）就已經在燒錢了，根本沒有多餘的現金可以去談投資未來。

這種負數是一個極度危險的警訊，代表公司的商業模式可能出了問題，核心業務正在失血。如果這種情況持續下去，公司就只能不斷地靠借錢或跟股東要錢（融資）來續命，這條路很難走得長久。

## 思考自由現金流數字代表什麼

不只是根據教學文章提到的資訊，套用公式計算、有沒有達成某指標，而是要當作自己「真的在看一間企業」，思考每項數字代表什麼狀況、會如何影響未來，當作自己真的是企業合夥人。

作為合夥人，你關心的不再只是損益表上的業績，甚至不只是現金流量表上的數字，而是你得透過財務報表審視這家公司有沒有正在為你的投資做合理的努力！
