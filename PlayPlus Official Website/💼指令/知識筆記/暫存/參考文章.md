# LINE Chatbot 聊天機器人的起源

LINE 最開始是一款提供人與人對話的聊天應用程式，當 LINE 開始普及化，就出現越來越多商業行為，像是客服或販賣商品，而這些商業行為可以自動化，也就是您在聊天視窗諮詢客服或購買商品，其實對方是機器人。

LINE 從 2016/04/07 開始提供「LINE Chatbot API」帳號申請，技術圈的人可以開始在 LINE 平台開發聊天機器人的多元應用。於是大家腦筋動很快，開始普遍應用到客服系統、天氣預報、股市資訊、販售商品…等等。而且 LINE 機器人不只是１對１聊天，擺在群組裡面也可以使用，達到多人互動的效果，像是群組團購或多人小遊戲等等。

## 常見的幾種操作類型

先知道有哪些常見的操作類型，對於 LINE 聊天機器人如何應用，你會很快有概念。

1. **確認型**： 回應簡單的二選一，是或否。
2. **按鍵型**： 結合圖片、文字及其他觸動按鈕，點擊後產生相應動作，例如拋出文字圖片訊息、網址、LINE頁面…等等。
3. **橫向捲軸型**： 文字圖片或按鈕等多個訊息組合成橫向捲動的方式呈現，使更多資訊直覺排列。
4. **文字指令型**： 輸入系統設定的文字指令呼叫動作，例如輸入 menu 出現菜單。
5. **圖文選單**： 畫面下方的固定按鈕，點擊後開啟相應動作，例如開啟會員功能、開啟工作清單。
6. **Liff頁面**： 等同於在 LINE 裡面開啟網頁，可以做比較複雜的互動，例如填寫表單。但是不需要跳出 LINE 外面即可操作。
7. **標籤與分眾推播**： 可以針對不同的族群推播不同的訊息。

## LINE 聊天機器人有哪些優勢？

LINE Chatbot 聊天機器人比較屬於「輔助」的性質，以電商舉例，若您原本已經是正在營運的電商，有一定數量的會員已經知道您能夠提供的服務與商品，換言之就是已稍微瞭解您的品牌是在做什麼的，此時將 LINE Chatbot 打造成客服系統或是提供會員快速下單，會員就知道自己為什麼要使用這個機器人，可以在上面買到什麼或做些什麼。

如果是直接以 LINE 機器人作為創業，要把自己的品牌描述清楚，難度就會比較高一些，因為網站或 APP 能夠呈現的資訊比較豐富，而機器人是精準提供服務，兩者屬性有些差異，但並非完全不可行，只要概念夠清楚，能夠簡單使用少數資訊把您要做的事情描述完畢，也可以達到同樣效果。例如便當店自取或外送下單系統，可以充分發揮高黏著度的優勢，而且對於使用者來說也不難懂。

還有對多數企業主來說，成本考量也是個優勢，如果您想做的服務比較簡單但是需要針對手機族群，LINE Chatbot 就能夠清楚呈現您所需要的動線以及想表達的事情，相較於 APP 造價成本會比較低一些唷。而且不用考慮針對各版本手機的相容性問題，因為 LINE 都幫我們做好了。

1. **高黏著度**： 台灣多數智慧型手機使用者幾乎每天都會使用 LINE。
2. **容易上手**： 不用重新學習如何使用。
3. **降低使用障礙**： 不用另外安裝 APP，也不用再註冊新會員，這兩件事對於很多人來說是大麻煩，而 LINE 機器人可以直接繞過去。
4. **多人互動**： 可以把大家集合在一個群組內共同使用功能。
5. **較低成本**： 成本低於於 APP。
6. **相容性高**： 能夠正常使用 LINE 的手機版本都沒問題。
