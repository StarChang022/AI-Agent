你參考我提供的[[暫存/說明.md]]和[[暫存/參考文章.md]]後，依照[[暫存/✅知識筆記文章規劃書.md]]的規劃開始撰寫文章，將產出結果於[[暫存]]資料夾下新增（或覆寫）「✅知識筆記.md」文件。

寫作時請遵守「# 寫作格式」及「# 額外規定」。

# 寫作格式

1. 文章最前面列出「## MetaTag」（不需要寫下標題「## MetaTag」）。
2. 每個段落一定要先寫下內文，才可以再接續子段落。例如 H2 段落必須要先寫內文，才可以再接續 H3 段落。
3. 最後一個段落一定要是「## 總結」。

## MetaTag

| 項目 | 值 |
| --- | --- |
| 標籤 | xxx |
| Description | xxx |
| Keywords | xxx |
| URL | xxx |

- 標籤（從**談談數位轉型**、**設計師邏輯**、**釐清需求**、**決策與規劃**、**觀點分享**、**工具與資源應用**、**趨勢觀察**、**其他**選出適合的標籤，數量不僅限於1個。）
- Description
- Keywords
- URL（網址結尾）

# 額外規定

1. 避免過度使用「不是…而是…」。
2. 減少使用「從…到…」。
3. 減少破折號。
