# 指示

1. 你會在檔案裡面看到被註解的「## 列表範例格式」，請複製範例格式並填入「我指定的 md 檔案」的內容。複製後的段落要放在「## 列表範例格式」的下方。如果當前指令要同時新增多篇文章，必須依照日期新舊排序，新的日期在上面；如果多篇文章的日期相同，則任意排序。

2. 其中「UrlName」填入「我指定的 md 檔案」上方的 URL 值（例如 [[20260311-傳統廚具大廠也需要數位轉型嗎？看櫻花集團如何以數據與體驗創造百億營收.md]] 就是填入「20260311-digital-transformation-sakura」）。

3. 其中「ArticleName」填入「我指定的 md 檔案」的檔案名稱，但是要去除掉開頭的日期格式（例如 [[20260311-傳統廚具大廠也需要數位轉型嗎？看櫻花集團如何以數據與體驗創造百億營收.md]] 就是填入「傳統廚具大廠也需要數位轉型嗎？看櫻花集團如何以數據與體驗創造百億營收」）。

4. 其中的「標籤」填入「我指定的 md 檔案」上方的標籤值，半形逗號表示區隔多個標籤，根據我的標籤數量新增標籤 <div>（例如 [[20260311-傳統廚具大廠也需要數位轉型嗎？看櫻花集團如何以數據與體驗創造百億營收.md]] 就是新增「談談數位轉型」和「觀點分享」兩個標籤）。

5. 承上，「<div class="col-sm-6 col-md-4 col-lg-3">」的 Class 要加上「<div class="filter column gap-micro centre">」的對應「data-filter」。

6. 其中「日期」填入「我指定的 md 檔案」的檔案名稱的開頭日期，並以 YYYY/MM/DD 格式呈現（例如 [[20260311-傳統廚具大廠也需要數位轉型嗎？看櫻花集團如何以數據與體驗創造百億營收.md]] 就是填入「2026/03/11」）。
