# 任務

自動讀取 [[冷郵件對象/名單副本.csv]] 的「官方網站」欄位與「來源」欄位。腳本將綜合運用 OSINT 技巧（包含：直接爬取 104 公司頁面、WHOIS 註冊反查、以及使用 Google Dorks 進階搜尋社群與官網）來查找符合的信箱，最終回填至 CSV。

執行時請參考 `# 使用步驟` 啟動爬蟲腳本。

# 使用步驟

1. 在終端機執行：
   ```bash
   python3 ⚙️參數設定/crawler_104_python/crawler_104_find_email.py
   ```
2. 腳本會自動讀取 [[冷郵件對象/名單副本.csv]]，解析網域後，依次執行三大 OSINT 查找策略：
   * 從「104 人力銀行」公司簡介頁面抓取公開 Email。
   * 從 Mac 系統終端機執行 `whois` 查詢域名註冊人聯絡信箱（並過濾註冊商代理信箱）。
   * 使用 Google Dorks 進階語法（如 Facebook 專頁或官網站內搜尋）與 DuckDuckGo 搜尋「@ + 網域」。
3. 腳本執行規則：
   - 找到 1 筆：直接填入「聯絡人信箱」欄位，並自動判斷是否為官方信箱來填寫「聯絡人名稱」。
   - 找到 2 筆以上：自動複製該列公司資料對應筆數，再分別填入對應的信箱與名稱。
   - 找不到：該列維持原狀保留。
