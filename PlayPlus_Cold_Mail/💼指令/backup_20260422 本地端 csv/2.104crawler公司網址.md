# 任務

自動進入 [[冷郵件對象/名單副本.csv]] 的「來源」欄位的網址，並找出「公司網址」寫回 CSV。

執行時請參考 `# 使用步驟` 啟動爬蟲腳本。

# 使用步驟

1. 在終端機執行：
   ```bash
   cd ⚙️參數設定/crawler_104_python
   python3 crawler_104_company_url.py
   ```
2. 腳本執行前會自動備份 [[冷郵件對象/名單副本.csv]]（備份檔名格式：`名單副本_backup_YYYYMMDD_HHMMSS.csv`）。
3. 腳本會逐列讀取 [[冷郵件對象/名單副本.csv]] 的「來源」欄位，使用 Playwright 進入 104 公司頁面尋找官方網站。
4. 腳本執行規則：
   - **官方網站已填寫**：略過該列，不重複處理。
   - **來源欄為空**：略過該列，保留原始資料。
   - **找到公司網址**：將 URL 寫入「官方網站」欄位，保留該列。
   - **找不到公司網址**：刪除該列（直接移除，無法復原，請確認備份存在）。
5. 腳本完成後結果直接回存至 [[冷郵件對象/名單副本.csv]]。
