# 任務

從 104 人力銀行公司搜尋頁面，收集「公司名稱」與「公司頁面連結」，並寫入 [[冷郵件對象/名單副本.csv]]：

- 公司名稱 → 「公司品牌簡稱」欄位
- 公司頁面連結 → 「來源」欄位

執行時請注意 `# 注意事項` 及記得必須遵照 `# 欄位` 的規定！

# 使用步驟

1. 打開 [[⌚️暫存/104_early_list.csv]]，確認或編輯要爬取的頁面網址（每行一個 URL，第一列為標題列，從第二列起填入網址）
2. 在終端機切換到腳本目錄後執行：
   ```bash
   cd ⚙️參數設定/crawler_104_python
   python3 crawler_104_early_list.py
   ```
3. 腳本會自動讀取 [[⌚️暫存/104_early_list.csv]] 的網址，並將結果附加寫入 [[冷郵件對象/名單副本.csv]]，略過已存在的網址（不重複）
4. 腳本執行完畢後，請讀取最新的 [[冷郵件對象/名單副本.csv]]。
5. 運用你的語言理解能力，將「公司品牌簡稱」欄位中過長的全名進行簡化，去除「股份有限公司、科技有限公司」等法律實體後綴，並保留最核心的「品牌名稱」（例如「懋霖生化科技有限公司」改為「懋霖生化」、「i good Point 好事交易所_綠籽文創股份有限公司」改為「i good Point」、「GODA夠搭選貨商店_夠搭有限公司」改為「GODA」、「穎德國際股份有限公司」改為「穎德國際」）。
6. 將整理好的品牌短名稱覆寫回 [[冷郵件對象/名單副本.csv]] 檔案中。

# 注意事項

- 腳本從 [[⌚️暫存/104_early_list.csv]] 讀取網址，**不需要**修改腳本本身。新增或移除頁面請直接編輯 CSV。
- [[⌚️暫存/104_early_list.csv]] 格式：第一列為標題列（`url`），第二列起每列一個 104 公司搜尋頁網址。
- 104 搜尋頁為 JavaScript 動態渲染，目前腳本已使用 Playwright 版本。
- 每次執行只會附加新資料，不會覆蓋現有記錄（以「來源」欄 URL 去重）。
