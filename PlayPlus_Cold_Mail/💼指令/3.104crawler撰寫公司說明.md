此指令搭配 [[⚙️參數設定/crawler_104_python/crawler_104_company_profile.py]] 使用，請完成 `# 任務` 中的任務。

# 任務

1. 執行爬蟲腳本抓取 104 頁面的公司簡介與產品資訊。
2. 爬蟲執行完畢後，會自動將擷取到的「簡介文本」暫存為 json 檔案。
3. 由 AI Agent 讀取該 json 檔案，根據內容，為每一間公司歸納總結出大約 50 ~ 100 字以內的一句話公司簡介（請使用繁體中文、專業口吻）。
4. 將撰寫好的說明寫回 [[冷郵件對象/名單副本.csv]] 的「說明」欄位。
5. 處理完畢後，刪除暫存的 json 檔案。

# 使用步驟

1. 在終端機執行：
   ```bash
   python3 ⚙️參數設定/crawler_104_python/crawler_104_company_profile.py
   ```
2. 等待腳本產生 `⚙️參數設定/crawler_104_python/temp_profiles.json`。
3. 讀取該 JSON，歸納總結後，寫回 `名單副本.csv` 的「說明」欄位。
