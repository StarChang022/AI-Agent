# 任務

1. 執行爬蟲腳本抓取 104 頁面的公司簡介與產品資訊。
2. 腳本執行時會自動進行規則式摘要，並將初步說明直接寫入 Google Sheets 的『名單副本』分頁「說明」欄位。
3. 腳本同時會將 raw 原文暫存為 `temp_profiles.json`，供 AI Agent 進行更細緻的精緻化摘要。
4. 如需 AI Agent 精緻化摘要：讀取 `temp_profiles.json`，根據內容為每一間公司歸納總結出大約 50 ~ 100 字以內的一句話公司簡介（請使用繁體中文、專業口吻），再覆寫回 Google Sheets 的『名單副本』分頁「說明」欄位。
5. 處理完畢後，刪除暫存的 `temp_profiles.json`。

# 使用步驟

1. 在終端機執行：
   ```bash
   python3 ⚙️參數設定/crawler_104_python/crawler_104_company_profile.py
   ```
2. 腳本會自動從 Google Sheets 的『名單副本』分頁讀取「來源」欄位，進入 104 頁面抓取公司資訊。
3. 腳本執行完畢後：
   - 初步說明已自動寫回 Google Sheets。
   - `temp_profiles.json` 儲存在 `⚙️參數設定/crawler_104_python/` 目錄下，供 AI Agent 精緻化使用。
4. （可選）由 AI Agent 讀取 `temp_profiles.json`，精緻化後再次覆寫「說明」欄位。
