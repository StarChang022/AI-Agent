現在要開始撰寫每間公司的公司說明，閱讀104頁面的資訊後，撰寫該公司的相關說明。

此指令搭配 [[⚙️參數設定/crawler_104_python/crawler_104_company_profile.py]] 使用，請完成 `# 任務` 中的任務。

# 任務

1. 執行爬蟲腳本抓取 104 頁面的公司簡介與主要商品資訊。
2. 爬蟲執行完畢後，會自動將擷取到的「簡介文本」暫存為 [[⚙️參數設定/crawler_104_python/temp_profiles.json]]。
3. 由 AI Agent 讀取該 JSON 檔案，根據內容，為每一間公司歸納總結出大約 50 ~ 100 字以內的一句話公司簡介（請使用繁體中文、專業口吻）。
4. 將撰寫好的說明，透過 `write_profiles_to_sheet(profiles_dict)` 函式或直接編輯，寫回 Google Sheets [[名單副本]] 的「說明」欄位（**不可覆蓋已有說明的列**）。
5. 處理完畢後，刪除暫存的 [[⚙️參數設定/crawler_104_python/temp_profiles.json]]。

# 使用步驟

1. 在終端機執行：
   ```bash
   cd ⚙️參數設定/crawler_104_python
   python3 crawler_104_company_profile.py
   ```
2. 腳本執行規則：
   - 從 Google Sheets [[名單副本]] 讀取所有列的「來源」欄位（104 公司頁面 URL）。
   - 若「來源」欄為空，略過該列。
   - 抓取每間公司的「公司簡介」（`.profile__desc`）與「主要商品」（`.product__desc`），並截取前 1000 字。
   - 若兩者皆找不到，則嘗試抓取頁面所有長度超過 30 字的 `<p>` 標籤段落作為備用。
3. 等待腳本產生 `⚙️參數設定/crawler_104_python/temp_profiles.json`。
4. 讀取該 JSON 檔案（格式：`{"公司品牌簡稱": "原始文本", ...}`），AI 逐一歸納後，寫回 Google Sheets 的「說明」欄位。
5. 刪除 `temp_profiles.json`。

# 欄位

Google Sheets 『名單副本』分頁共 9 欄（A~I）：

- **A 公司品牌簡稱**：維持不動。
- **B 序號**：維持不動。
- **C 官方網站**：維持不動。
- **D 產業**：維持不動。
- **E 員工人數**：維持不動。
- **F email**：維持不動。
- **G 聯絡人名稱**：維持不動。
- **H 來源**：維持不動。
- **I 說明**：依照 `# 任務` 填寫（AI 撰寫 50~100 字繁體中文公司說明）。
