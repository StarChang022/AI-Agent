現在要開始查找每間公司網域的信箱，透過爬蟲查找，收集有效的 Email 名單。

此指令搭配 [[⚙️參數設定/crawler_104_python/crawler_104_find_email.py]] 使用，請完成 `# 任務` 中的任務。

# 任務

自動讀取 Google Sheets [[名單副本]] 的「官方網站」欄位與「來源」欄位。腳本將綜合運用 OSINT 技巧，依序執行四種策略查找符合網域的信箱，最終回填至 Google Sheets。

執行時請參考 `# 使用步驟` 啟動爬蟲腳本。

# 使用步驟

1. 在終端機執行：
   ```bash
   cd ⚙️參數設定/crawler_104_python
   python3 crawler_104_find_email.py
   ```
2. 腳本會從 Google Sheets 讀取 [[名單副本]] 分頁的所有資料，解析「官方網站」欄位的網域後，依次執行四大 OSINT 查找策略：
   1. **104 公司頁面**：直接爬取「來源」欄位的 104 公司簡介頁面，抓取公開 Email。
   2. **WHOIS 查詢**：執行 `whois` 指令查詢域名註冊人聯絡信箱（並過濾 abuse、privacy、host、twnoc、twnic、domain、admin 等代理信箱）。
   3. **DuckDuckGo 搜尋**：搜尋「@網域」關鍵字，從搜尋結果抓取 Email。
   4. **Google Dorks**：使用進階語法（如 Facebook 專頁站內搜尋、官網站內 email 搜尋）抓取 Email。
3. 腳本執行規則：
   - **官方網站欄為空**：略過該列，保留原始資料。
   - **email 欄已有值**：略過該列，不重複處理。
   - **找到 1 筆**：直接填入「email」欄位，並自動判斷是否為官方信箱（info@, contact@, service@ 等）來填寫「聯絡人名稱」（官方信箱填「官方」，具名信箱嘗試從前綴推測姓名）。
   - **找到 2 筆以上**：複製該列公司資料對應筆數，再分別填入對應的信箱與名稱（每筆 Email 一列）。
   - **找不到**：該列維持原狀保留。
4. 腳本完成後，結果會**完整覆寫**回 Google Sheets [[名單副本]] 分頁：
   https://docs.google.com/spreadsheets/d/14H99Ks5UFbdNnM9OoNQ2XWoVz4UHyp2QK0GiIym_1pE/edit?gid=1168472169#gid=1168472169

# 欄位

Google Sheets 『名單副本』分頁共 9 欄（A~I），以下皆由腳本自動處理，無須手動編輯：

- **A 公司品牌簡稱**：維持不動。
- **B 序號**：維持不動。
- **C 官方網站**：維持不動。
- **D 產業**：維持不動。
- **E 員工人數**：維持不動。
- **F email**：腳本自動填寫（找到的公司網域信箱）。
- **G 聯絡人名稱**：腳本自動判斷填寫（官方信箱填「官方」；具名信箱填推測名稱）。
- **H 來源**：維持不動。
- **I 說明**：維持不動。
