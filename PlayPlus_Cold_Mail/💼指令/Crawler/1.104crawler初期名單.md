# 任務

從 104 人力銀行公司搜尋頁面，收集「公司名稱」與「公司頁面連結」，並**覆寫**至 [[冷郵件對象/104初期名單.csv]]：

- 公司頁面連結 → `104頁面連結` 欄位

# 使用步驟

1. 打開 [[⌚️暫存/104_early_list.csv]]，確認或編輯要爬取的頁面網址（第一列為標題列 `url`，從第二列起填入網址）
2. 在終端機切換到腳本目錄後執行：
   ```bash
   cd ⚙️參數設定/crawler_104_python
   python3 crawler_104_early_list.py
   ```
3. 腳本會自動讀取 [[⌚️暫存/104_early_list.csv]] 的網址，將所有結果**覆寫**至 [[冷郵件對象/104初期名單.csv]]（同一批次執行內自動去重）
4. 腳本執行完畢後，請前往 [[冷郵件對象/104初期名單.csv]] 確認結果

# 注意事項

- 腳本從 [[⌚️暫存/104_early_list.csv]] 讀取爬取目標，**不需要**修改腳本本身。新增或移除頁面請直接編輯 CSV。
- 每次執行會**完整覆寫** [[冷郵件對象/104初期名單.csv]]，原有資料將被取代。
- 104 搜尋頁為 JavaScript 動態渲染，腳本使用 Playwright 版本。
- 同一批次執行過程中會自動去重（以公司頁面 URL 為準）。

# 欄位
