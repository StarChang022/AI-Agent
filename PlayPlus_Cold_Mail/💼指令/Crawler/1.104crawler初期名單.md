# 任務

從 104 人力銀行公司搜尋頁面，收集「公司名稱」與「公司頁面連結」，並寫入 Google Sheets 的 [[名單副本]] 分頁：

- 公司名稱 → 「公司品牌簡稱」欄位（A 欄）
- 公司頁面連結 → 「來源」欄位（H 欄）

執行時請注意 `# 注意事項` 及記得必須遵照 `# 欄位` 的規定！

# 使用步驟

1. 打開 [[⌚️暫存/104_early_list.csv]]，確認或編輯要爬取的頁面網址（第一列為標題列 `url`，從第二列起填入網址）
2. 在終端機切換到腳本目錄後執行：
   ```bash
   cd ⚙️參數設定/crawler_104_python
   python3 crawler_104_early_list.py
   ```
3. 腳本會自動讀取 [[⌚️暫存/104_early_list.csv]] 的網址，將結果**附加**寫入 Google Sheets，略過已存在的網址（以「來源」欄 URL 去重）
4. 腳本執行完畢後，請前往 Google Sheets 查看最新的 [[名單副本]] 分頁：
   https://docs.google.com/spreadsheets/d/14H99Ks5UFbdNnM9OoNQ2XWoVz4UHyp2QK0GiIym_1pE/edit?gid=1168472169#gid=1168472169
5. 運用你的語言理解能力，將「公司品牌簡稱」欄位中過長的全名進行簡化，去除「股份有限公司、科技有限公司」等法律實體後綴，並保留最核心的「品牌名稱」（例如「懋霖生化科技有限公司」改為「懋霖生化」、「i good Point 好事交易所_綠籽文創股份有限公司」改為「i good Point」、「GODA夠搭選貨商店_夠搭有限公司」改為「GODA」、「穎德國際股份有限公司」改為「穎德國際」）。
6. 將整理好的品牌短名稱覆寫回 Google Sheets 的「公司品牌簡稱」欄位。

# 注意事項

- 腳本從 [[⌚️暫存/104_early_list.csv]] 讀取爬取目標，**不需要**修改腳本本身。新增或移除頁面請直接編輯 CSV。
- 資料寫入目的地為 Google Sheets，使用 Service Account 金鑰 [[⚙️參數設定/crawler_api/eternal-skyline-494002-j8-356884d3e786.json]] 進行認證。
- 104 搜尋頁為 JavaScript 動態渲染，腳本使用 Playwright 版本。
- 每次執行只會附加新資料，不會覆蓋現有記錄（以「來源」欄 URL 去重）。

# 欄位

Google Sheets 『名單副本』分頁共 9 欄（A~I）：

- **A 公司品牌簡稱**：依照 `# 任務` 填寫（腳本填入全名，執行後由 AI 整理為品牌簡稱）。
- **B 序號**：維持不動。
- **C 官方網站**：維持不動。
- **D 產業**：維持不動。
- **E 員工人數**：維持不動。
- **F email**：維持不動。
- **G 聯絡人名稱**：維持不動。
- **H 來源**：依照 `# 任務` 填寫（腳本自動填入 104 公司頁面 URL）。
- **I 說明**：維持不動。
