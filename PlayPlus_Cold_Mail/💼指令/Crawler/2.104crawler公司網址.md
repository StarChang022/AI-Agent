# 任務

自動進入 Google Sheets [[名單副本]] 分頁的「來源」欄位網址，並找出「公司網址」寫回 Google Sheets。

執行時請參考 `# 使用步驟` 啟動爬蟲腳本。

# 使用步驟

1. 在終端機執行：
   ```bash
   cd ⚙️參數設定/crawler_104_python
   python3 crawler_104_company_url.py
   ```
2. 腳本會自動從 Google Sheets 讀取 [[名單副本]] 分頁的所有資料，使用 Playwright 進入 104 公司頁面尋找官方網站。
3. 腳本執行規則：
   - **官方網站已填寫**：略過該列，不重複處理。
   - **來源欄為空**：略過該列，保留原始資料。
   - **找到公司網址**：將 URL 寫入「官方網站」欄位，保留該列。
   - **找不到公司網址**：保留該列待查（不刪除資料）。
4. 腳本完成後，結果會**完整覆寫**回 Google Sheets [[名單副本]] 分頁：
   https://docs.google.com/spreadsheets/d/14H99Ks5UFbdNnM9OoNQ2XWoVz4UHyp2QK0GiIym_1pE/edit?gid=1168472169#gid=1168472169

# 欄位

Google Sheets 『名單副本』分頁共 9 欄（A~I），以下皆由腳本自動處理，無須手動編輯：

- **A 公司品牌簡稱**：維持不動。
- **B 序號**：維持不動。
- **C 官方網站**：腳本自動填寫公司官方網站 URL。
- **D 產業**：維持不動。
- **E 員工人數**：維持不動。
- **F email**：維持不動。
- **G 聯絡人名稱**：維持不動。
- **H 來源**：維持不動。
- **I 說明**：維持不動。
