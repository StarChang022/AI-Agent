# 架構規格

規格為 Table 格式。

- 「**模組**」是由多個具備基礎功能的**頁面**、**後台**或**系統**組成一個具有情境意義的獨立單元。
- 「**頁面**」是指網站或系統的前台頁面。
- 「**後台**」是指管理者或內部人員使用的後台管理系統，通常用於管理網站或系統的資料。
- 「**系統**」是指寫在系統內的邏輯規則，不會被使用者及管理者從系統介面上操作。
