以下為當前舊版本系統的管理項頁面及欄位。

## 副管理者列表

新增刪除編輯多個副管理者。

- 帳號
- 暱稱

## 副管理者編輯

編輯副管理者的資訊。

- 帳號
- 暱稱
- 密碼
- 確認密碼

## 個人資料

用於管理者編輯自己的資料。

- 帳號
- 暱稱
- 新密碼
- 確認新密碼

## 活動管理列表

新增刪除編輯多個活動。

- 活動日期
- 活動名稱
- 廠商名稱
- 聯絡人
- 管理員

## 活動編輯

編輯管理活動的概略資訊。

- 活動名稱／活動日期
- 廠商名稱／聯絡人／電子郵件／聯絡電話／公司地址
- 手動抽獎／管理員／手動抽獎裝置管理
- 關卡啟用／關卡數量／最低通關
- 點數啟用／報到點數／公益點數啟用／報到公益點數

## 活動內頁

顯示活動概略資訊，包含活動名稱、參加人數、報名人數、已報到人數、已離場人數等，以及列出參加者總名單。

可以在這個頁面操作新增刪除編輯參加者（及 Excel 大量匯入參加者），控制本活動要使用的**參加者資料欄位**。

補充說明**參加者資料欄位**，系統預先寫好**隸屬公司名稱**、**員工編號**、**姓名**、**年資**、**隸屬廠區**、**部門**、**手機電話**、**公司電話**、**電子郵件一**、**電子郵件二**、**眷屬人數**、**點數**、**公益點數**這幾個欄位，後續根據每次活動決定要開放哪些欄位。

- 序號（系統流水號）
- 員工編號
- 年資
- 姓名
- 部門
- E-mail
- 是否登入（顯示是否曾經登入過）
- 不抽獎（被標註不抽獎者無法參加抽獎）
- 報到狀態（顯示狀態以及管理者手動切換）
- 離場狀態（顯示狀態以及管理者手動切換）

### 參加者詳細資料

從**活動內頁**點擊查看參加者，顯示詳細資料及報到 QR Code。除了以下列出的欄位之外，還得顯示點數使用狀況，包含剩餘點數、使用日期時間、使用商家名稱、使用點數。

- 隸屬公司名稱
- 部門/單位
- 員工編號
- 姓名
- 工作年資
- 隸屬廠區
- 公司電話
- 手機電話
- 電子信箱一
- 電子信箱二
- 是否搭乘接駁車
- 是否攜帶眷屬
- 眷屬人數
- 不參加抽獎
- 點數
- 公益點數

### 獎項管理

從**活動內頁**點擊管理所有獎項，管理該活動的獎項，可以新增刪除編輯多個獎項。

- 獎品名稱
- 獎品照片
- 獎品名稱
- 可重複抽獎（若有打勾則表示已得獎的參加者可再次中獎）
- 全體人員皆可抽（若有打勾則表示無論參加者有無報到皆可中獎）
- 年資限定（符合年資的參加者才能被抽獎）
- 中獎人數（該獎項可以有幾位得獎人）
- 啟用（有打勾才表示該獎項可被使用）

### 獎項編輯

從**獎項管理**內新增或編輯某個獎項。

- 獎項名稱
- 照片上傳
- 獎品名稱
- 抽獎順序
- 得獎人數
- 抽獎年資
- 可重複抽獎
- 啟用狀態

### 文件匯出

從**活動內頁**點擊查看相關的文件及網址。其中特別說明**網址**，當前舊版系統需要使用大量的 QR Code 進行流程，所以各種功能都會產生「網址」。

- QR Code 取得網址（參加者從這邊輸入資料並取得 QR Code 供活動現場人員掃描報到）
- 闖關登入網址（參加者從這邊拿取每個關卡的 QR Code 供活動現場關主掃描）
- 商家登入網址（商家登入取得收款 QR Code 及查看統計）
- 客戶中獎名單取得網址（查看中獎結果）
- 報到名單 QRCode 下載（直接將所有報到 QR Code 列出）
- 手動抽獎名單 QRCode 下載（活動現場人員為每個獎項掃描得獎者）
- 登入名單（列出有哪些參加者已登入）
- 報到人員名單（列出有哪些參加者已報到）
- 離場人員名單（列出有哪些參加者已離場）
- 所有中獎者清單（列出獎項及得獎者名單）
- 所有領獎者清單（列出得獎者是否已領獎）
- 所有通關者清單（列出每個關卡的通關者）
- 所有店家付費明細（列出所有商家的營收統計及點擊付款歷史紀錄）

### 關卡清單

從**活動內頁**點擊管理關卡，可以新增刪除編輯多個關卡。

- 序號
- 關卡名稱
- 建立者

### 店家清單

從**活動內頁**點擊管理攤商，可以新增刪除編輯多個攤商。

- 序號
- 店家代號
- 店家名稱
- 建立者

### 店家編輯

從**活動內頁**點擊編輯攤商資料。

- 店家編號
- 店家名稱
- 密碼
- 確認密碼
- 商店類型（一般攤商／公益攤商）
