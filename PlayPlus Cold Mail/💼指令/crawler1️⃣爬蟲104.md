根據以下「# 欄位規則」，從 [[冷郵件對象/名單副本.csv]] 的「來源」欄位的 URL 爬取相應的欄位資料，直接覆寫於 csv 檔。

先檢查 URL 裡面是否有「公司網址」資料，如果有才執行；如果沒有則無需爬取該公司資料。

如果有需要 Allow 權限開啟瀏覽器，一律直接 Approval。

# 欄位規則

- 公司品牌簡稱： 填寫 URL 裡面的公司品牌簡稱，不需要寫完整的有限公司或股份有限公司之類的。例如永恆包裝有限公司就填寫「永恆包裝」。
- 序號： 維持現狀，不准填寫也不准刪除。
- 官方網站： 如果 URL 裡面有「公司網址」資料，則填入連結；如果沒有則維持現狀。
- 產業： 維持現狀，不准填寫也不准刪除。
- 員工人數： 維持現狀，不准填寫也不准刪除。
- 聯絡人信箱： 維持現狀，不准填寫也不准刪除。
- 聯絡人名稱： 維持現狀，不准填寫也不准刪除。
- 聯絡人職稱： 維持現狀，不准填寫也不准刪除。
- 來源： 從來源 URL 爬取資料。
- 徵才職缺名稱： 如果 URL 裡面有工作機會，則填入職缺名稱（多個名稱之間以頓號分隔）；如果沒有則維持現狀。
- 說明： 根據 URL 的介紹，為此公司撰寫基本介紹。
- 備註： 如果還有什麼重要資訊，請補充；如果沒有則維持現狀。
