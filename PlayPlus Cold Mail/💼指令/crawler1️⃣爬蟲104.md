# 腳本內建之邏輯規則 (供參考)

腳本會自動遵循以下規則更新檔案：

- 公司品牌簡稱： 自動填寫 URL 裡面的公司品牌簡稱，並去除「股份有限公司」等字眼。例如永恆包裝有限公司就填寫「永恆包裝」。
- 序號： 維持現狀，不准填寫也不准刪除。
- 官方網站： 如果 URL 裡面有「公司網址」資料，則填入連結；如果沒有則維持現狀。
- 產業： 維持現狀，不准填寫也不准刪除。
- 員工人數： 維持現狀，不准填寫也不准刪除。
- 聯絡人信箱： 維持現狀，不准填寫也不准刪除。
- 聯絡人名稱： 維持現狀，不准填寫也不准刪除。
- 聯絡人職稱： 維持現狀，不准填寫也不准刪除。
- 來源： 從來源 URL 爬取資料。
- 徵才職缺名稱： 如果 URL 裡面有工作機會，則填入職缺名稱（多個名稱之間以頓號分隔）；如果沒有則維持現狀。
- 說明： 根據 URL 裡面的內容介紹，為此公司撰寫基本介紹。
- 備註： 維持現狀，不准填寫也不准刪除。
